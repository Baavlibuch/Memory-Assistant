<p> É uma ferramenta poderosa que é usada pela maioria dos atletas mentais. Como é baseado em um sistema fonético, você memorizará facilmente todas as imagens e passará a utilizá-las em ação. </p>

<h4> O desafio dos números </h4>

<p> A razão pela qual os números são tão problemáticos para muitas pessoas é que elas são abstratas. Eles caracterizam uma quantidade de algo, mas não são nada que possamos ver, segurar ou sentir. Nós não podemos nem descrever o que eles significam. Claro, você pode tentar por um dígito muito baixo. Mas quando se trata de mais de dez você terá grandes problemas. A conclusão para atletas de memória é tornar os números mais concretos. </p>

<h4> Os diferentes sistemas de pinos </h4>

<p> Para manifestar um número, você precisa alterá-lo em outra coisa. Não importa em que, desde que você possa imaginar. Como temos um sistema decimal, precisamos traduzir pelo menos dez imagens para cada número. As listas que você cria a partir desses objetos são amplamente chamadas de “listas de pegadas”. Eu me referirei a esses objetos como “imagens” ou “mnemônicos”. Depois de memorizar essa lista, você poderá usá-la repetidamente e memorizar qualquer quantidade de dígitos (dependendo da quantidade de locais do seu Método de Localização). </p>

<p> Agora veremos alguns sistemas numéricos diferentes usados ​​por atletas de memória. A complexidade crescente é indicada pelo termo “Nível”. Um nível superior contém mais informações dentro de uma imagem e, portanto, é mais eficiente para a tarefa de memorização. Ao mesmo tempo, o esforço cresce para aprender o sistema. Um sistema de 1º nível pode ser aprendido dentro de cinco minutos. Para um sistema de 3º nível, é melhor prever algumas semanas. Depende do seu propósito que você deve usar. Para estudos, você pode estar bem com os sistemas menores. Se você quer se tornar um campeão de memória, você deve buscar maior complexidade. </p>

<p> Esta visão geral é apenas um exemplo dos sistemas mais populares. As possibilidades são muito maiores. Todo novato é aconselhado a ter em mente que sempre pode haver uma solução melhor para traduzir os números em algo mais memorável. </p>
